# Data Privacy - CCPA Compliance

## Overview

California Consumer Privacy Act compliance requirements.

## Requirements

### Right to Know
- Disclose what personal information is collected
- Explain how it's used and shared

### Right to Delete
- Provide mechanism to delete personal data
- Complete deletion within 45 days

### Right to Opt-Out
- Allow opt-out of data selling
- Provide "Do Not Sell" link

### Right to Non-Discrimination
- Cannot deny service for exercising rights

## Data Categories

### Lead Data
- Name, email, phone
- Property interests
- Communication history

### Analytics Data
- Page views
- Session data
- IP addresses

## Implementation

- Privacy policy page
- Data deletion request form
- Cookie consent banner
- Analytics opt-out option
