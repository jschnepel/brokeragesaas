# ARMLS Compliance Requirements

## Display Requirements

### Required Disclaimers
- MLS logo must be displayed
- Last updated timestamp required
- Broker reciprocity notice required

### Data Attribution
- Source attribution on all listings
- Listing agent/office credit

## Data Usage

### Rate Limits
- Respect API rate limits
- Cache data appropriately
- Update data at required intervals

### Data Retention
- Follow ARMLS data retention policies
- Remove sold listings per guidelines

## IDX Requirements

- Proper registration required
- Broker consent documentation
- User registration for detailed views (optional)

## Implementation Checklist

- [ ] Display MLS logo
- [ ] Show last updated timestamp
- [ ] Include broker reciprocity
- [ ] Attribute listing sources
- [ ] Implement rate limiting
- [ ] Configure data sync intervals
